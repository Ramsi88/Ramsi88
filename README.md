## Bio

Ramzi Bougammoura alias/aka Ramsi.

The world, technology, arts and communication has always fascinated me so I cultivated a passion to connect these realities.

In my past I worked several years as a certified hardware and software technician for Apple and as a teacher of audio engineering and sound design.

I work as a Software Engineer using differents hardware and software technologies to craft things and concepts and in the last years I focused my work within the blockchain sector, DLT and FinTech, developing a passion for cryptography and contributing to the development and integration of blockchain systems, multi-signature based custody protocols using HSMs, primarily on the Bitcoin protocol and the Ethereum ecosystem.

## Projects & Portfolio

- **[Trusty](https://github.com/Ramsi88/trusty)** EVM MultiSignature contracts and interface. ([DEMO](https://trusty-rms.vercel.app/))
  - [Trusty Interface](https://github.com/Ramsi88/trusty-app) Trusty App - Next.js App interface for contracts interactions.
  - [Trusty Conctracts](https://github.com/Ramsi88/trusty-contracts) Trusty Conctracts - Solidity MultiSignature contracts.
- **[HWI-client](https://github.com/Ramsi88/hwi-client)** (Private) JS Hardware Wallet Interface client for Ledger. Transaction and PSBTs sign and broadcast. Included `btc-signer` and `btc-tx-builder` packages. ([DEMO](https://hwi-client.vercel.app/))
- **[WEB-TUI](https://github.com/Ramsi88/web-tui)** (Private) Web Terminal User Interface made in Rust with Ed25519, Bitcoin and Ethereum wallet creation. ([DEMO](https://web-tui.vercel.app))
- **[RMS-TUI](https://github.com/Ramsi88/web-tui)** (Private) Terminal User Interface made in Rust with System Info, Networks connections tracking, SQLite DB interaction, File manager and other features.
- **[btc-signer](https://github.com/Ramsi88/btc-signer)** (Private Package) JS Bitcoin Wallet generator and Signer (ECDSA and Schnorr signing).
- **[btc-tx-builder](https://github.com/Ramsi88/btc-tx-builder)** (Private Package) JS Bitcoin Transaction and PSBT builder.
- **[rust-btc-wallet](https://github.com/Ramsi88/rust-btc-wallet)** (Private) Rust Bitcoin Wallet and Transaction builder.
- **[btc-wallet](https://ramsi.it)** (Private) Python Bitcoin wallet generator.
- **[eth-signer](https://github.com/Ramsi88/eth-signer)** (Private) Ethereum Wallet Signer and Transaction builder for ETH and ERC20 Tokens.
- **[rms-jc-client](https://github.com/Ramsi88/rms-jc-client)** (Private) Java Card tool for Applets install and Smart Card interactions within Secure Channel Protocl runtime in node.
